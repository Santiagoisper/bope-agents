# BOPE Agent Squad

8 specialized agents for Claude Code and Codex. Type `bope` and John orchestrates the team.

## Agents

| Agent | Model | Role |
|-------|-------|------|
| JOHN | sonnet | Orchestrator |
| SCOUT | haiku | Research & analysis |
| PIXEL | sonnet | Frontend (React, TypeScript, Tailwind) |
| FORGE | sonnet | Backend (Node.js, PostgreSQL/Neon, APIs) |
| HOUSE | haiku | QA & debugging |
| NEXUS | haiku | GitHub & deploy (gh, vercel) |
| CERBERUS | haiku | Security |
| SICARIO | haiku | Bulk execution & refactors |

## Install

### Mac / Linux
```bash
git clone https://github.com/Santiagoisper/bope-agents.git
cd bope-agents
bash install.sh
```

### Windows (PowerShell)
```powershell
git clone https://github.com/Santiagoisper/bope-agents.git
cd bope-agents
.\install.ps1
```

## Usage

**Claude Code**
```
bope
```

**Codex**
```
/bope
```
